Basic HTML Website

Elements to use

  • <head>
  • <body>
  • <h1>
  • <h2>
  • <p>
  • <ul>
  • <ol>
  • <dl>

Steps Needed

  1. Add <!doctype html> and </html> to the page
  2. Add <head> and </head> in between the previous tags
  3. Add <title> and </title> in between the head elements
  4. Write your title in between the title elements
  5. Directly under the closing head tag, add <body> and </body>
  6. Now you can write the content of your website! (Make sure to use headers, paragraphs, and lists!)
